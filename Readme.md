# ✈️ ATPLQ Discord Rich Presence

A Discord Rich Presence integration for ATPL Questions.

Display your:
- current subject
- study/exam mode
- question progress
- success rate
- session timer

directly on your Discord profile.

---

# 📸 Features

✅ Study Mode detection  
✅ Exam Mode detection  
✅ Live question tracking  
✅ Success rate tracking  
✅ Correct / Wrong answers  
✅ Session timer  
✅ Browsing menu detection  
✅ Discord Rich Presence integration  
✅ Chrome extension support

# 🚀 Installation

# 1. Download the project

Click:

```text
Code → Download ZIP
```

or clone:

```bash
git clone https://github.com/YOURNAME/ATPLQ-RichPresence.git
```

---

# 2. Install Node.js

Download Node.js:

https://nodejs.org/

Verify installation:

```bash
node -v
npm -v
```

---

# 3. Install server dependencies

Open terminal inside:

```text
server
```

Run:

```bash
npm install
```

---

# 4. Load the Chrome extension

Open Chrome and go to:

```text
chrome://extensions
```

Enable:

```text
Developer Mode
```

Click:

```text
Load unpacked
```

Select the:

```text
extension
```

folder.

---

# 5. Start the RPC

Double click:

```text
start.bat
```

---

# ✅ Done

Open:
- Discord
- Chrome
- ATPL Questions

and your Rich Presence will automatically update.
